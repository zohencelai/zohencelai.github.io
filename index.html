Get Started with Zohencel-AI ML Bot

Install the latest version using pip:

pip install zohencel-ai

Run the code in a new Python file:

from zohencel_ai.mlbot.bot import ZohencelmlBot
bot = ZohencelmlBot()
bot.run()
