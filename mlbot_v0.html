Introduction

Welcome to Zohencel AI ML Bot, the world's first 'Text to ML Model' chat application. Whether you're a beginner or an expert, our ML Bot simplifies the process of building, training, and deploying machine learning models through an intuitive conversational interface.

Powered by cutting-edge LLM models from Groq, the ML Bot automates every step of the machine learning pipeline, from data analysis to model deployment. Say goodbye to complex coding and hello to seamless AI development!

When you provide your data as input, the ML Bot performs a comprehensive analysis, including:

  • Missing Data Analysis: Identifies and handles missing values in your dataset.
  • Duplicated Data Analysis: Detects and removes duplicate entries.
  • Outlier Analysis: Identifies and manages outliers that could skew your results.
  • Skew Analysis: Checks for data skewness and applies transformations if necessary.
  • Encoding Analysis: Automatically encodes categorical variables for model compatibility.
  • Correlation Analysis: Analyzes relationships between features to identify key predictors.
  • Feature Scaling Analysis: Ensures all features are on a similar scale for better model performance.

In today's world, deep mathematical and statistical knowledge is often required to preprocess data effectively and achieve better accuracy in machine learning projects. The Zohencel AI ML Bot eliminates this complexity by providing a complete training and testing script, along with a detailed descriptive analysis of your data. This allows you to understand your data better and make informed decisions without worrying about the technical details.

Once the analysis is complete, you can download a zip file containing all the necessary scripts and files. Simply integrate these into your environment, and you're ready to go! I'm thrilled to offer these powerful features in the Community Edition of the ML Bot. I hope you enjoy using it and find it as revolutionary as I do. Use it wisely, and let's build amazing machine learning models together!

Features

  • Interactive Model Creation: The ML Bot interacts with you to understand your requirements, even if you're new to machine learning.
  • Automated Statistical Analysis: Performs statistical and descriptive analysis, suggesting best practices for model training.
  • ML Training Pipeline: Automatically creates and executes a machine learning pipeline, handling all data transformations.
  • Model and Script Generation: Saves the trained model as a .pkl file and generates training scripts with accuracy metrics.
  • Downloadable Files: Download models and associated files anytime from the Models tab.
  • Portability: Use generated training data and scripts in other environments, ready for API development.

Usage

Getting started with Zohencel AI ML Bot is as easy as having a conversation. Follow the steps below to unleash the power of AI:

Step 1: Install Zohencel AI

pip install zohencel-ai

Step 2: Run the ML Bot

from zohencel_ai.mlbot.bot import ZohencelmlBot
bot = ZohencelmlBot()
bot.run()

Optional: Use Your Groq API Key

For smooth operations and to avoid rate limits, we recommend creating your own Groq API key for free. You can also select the LLM model of your choice for machine learning model development.

from zohencel_ai.mlbot.bot import ZohencelmlBot
bot = ZohencelmlBot(
    groq_api_key='XXXXXX',  
    groq_model='llama-3.3-70b-versatile'  
)
bot.run()
